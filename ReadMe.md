# Better Transit View
A more usable Transit Overview for Cities II.
## Features:
- Compact design.
- View all of the same data as the vanilla transit overview.
- Use route tools while the UI is open. You can finally hide overlapping lines so you can edit the route you want.
- Toggle on or off a gray world map.
- Route lines are drawn much thicker as you zoom out, so you can see an overview of the routes across your map.
- Where two lines share the same stop, an indicator is drawn with the lines' colors.
